ESP NOW Communication System

ESP-NOW is a wireless communication protocol developed by Espressif that features short packet transmission. This protocol enables multiple devices to talk to each other without using Wi-Fi. This is a fast communication protocol that can be used to exchange small messages (up to 250 bytes) between ESP32 boards.

Multiple Sensors Data Read One Channel

The DS18B20 temperature sensor communicates using one-wire protocol and each sensor has a unique 64-bit serial code, so it can be easily read the temperature from multiple sensors using just one single Arduino digital Pin.

Data Logger

Its a multiple sensors data logger that can read data of temperature, pressure and flow of a system. Also user friendly dashboard is developed for these parameters with python and Qt designer. Here, Arduino nano is used as a micro-controller.
